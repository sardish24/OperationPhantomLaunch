# NOOR: The 2:00 AM Venting Protocol

> "Syllabus impossible. Mind shattered. Don't hold it in."

Noor is an adversarial, secure, anonymous voice-and-text venting application explicitly modeled for the **Operation Phantom Launch** strategy. It caters to highly distressed engineering and medical aspirants (specifically in hubs like Kota, India), providing them with a hyper-local, judgment-free space to vent out tension precisely when academic anxiety hits the hardest.

## Features

- **No Gyan, Just Venting:** No unsolicited advice, clinical therapy buzzwords, or preachy UI. Just an open space to let your thoughts out.
- **Audio Visualizer Protocol:** Features a warm, reactive audio visualizer hooked directly to the browser's Web Audio API to confirm active listening.
- **Ephemeral & Secure:** Everything spoken or typed is explicitly stored in active memory only. Using the "Burn & Close / End Session" sequence instantaneously destructs media streams, frees system resources, and wipes out all typed content.
- **Text Mode Fallback:** Complete anonymous text interface for strict covert operation if speaking aloud is not safe from roommates or parents.
- **Phantom Launch Aesthetics:** Vantablack background, CRT atmospheric noise overlays, and specific monospace typography that replicates a secure, "underground" terminal experience.

## Project Structure

- `index.html` — The structural layout, interactive canvas sections, and routing layout.
- `style.css` — High-fidelity, zero-dependency custom style sheet integrating modern CSS Grid, Flexbox, and keyframe animations.
- `script.js` — The core client logic governing Web Audio API, dynamic layout toggles, and stream destruction.

## Usage & Deployment

This application operates entirely horizontally using 100% client-side logic—ensuring there are zero backend logs, network requests, or tracker injections. 

To run this locally:
1. Open the project folder.
2. Double-click `index.html` to run the application in a local browser (a server is not required).
3. If you intend to use the Voice protocol, you must click "Allow" on the browser microphone permission prompt. 

## Design Philosophy

Noor exists to break the clinical loops of traditional mental wellness applications. It is not attempting to analyze, label, or heal; it is built strictly to absorb pressure. 

**No accounts. No tracking. No history. You speak into the void, the void listens, and then the void forgets.**
